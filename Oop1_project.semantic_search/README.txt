1. Introduction:
-----------------------------------------------------------------------
CTexTools is a corpus query and manipulation tool for performing tokenisation and sentencisation, 
annotating data, extracting frequency and word lists, and for searching and extracting collocations. 

Developed by the Centre for Text Technology (CTexT®), North-West University (Potchefstroom Campus) 
and funded by the Department of Arts and Culture, Government of South Africa.

CTexTools is free software: you can use it on as many computers as you wish, and even redistribute it. 
CTexTools is distributed in the hope that it will be useful, but without any warranty, even the 
implied warranty of merchantability or fitness for a particular purpose. 
We are certain that you will be satisfied with this product! 

For more information about the developers, please see http://humanities.nwu.ac.za/ctext

2. Required files:
-----------------------------------------------------------------------

The installation file for CTexTools:

	./Install_CTexTools.exe
	
_______________________________________________________________________
3. Installing:
-----------------------------------------------------------------------

Run "Install_CTexTools.exe" by double clicking on it *.
Follow the instructions and finish the installation.

* Right click and "Run as administrator" to install CTexTools to "C:\Program Files"
if you are not using an administrator account.

_______________________________________________________________________
4. Using CTexTools:
-----------------------------------------------------------------------

	1. Click on Start.
	2. Browse through "All Programs".
	3. Go to "CTexTools -> CTexTools"
	4. Clicking on the shortcut will launch the application.
	5. If asked for the administrator password, enter it and continue.

_______________________________________________________________________
5. Minimum System Requirements	
-----------------------------------------------------------------------

These are the recommended minimum system requirements for CTexTools to run.

	Operating System: 		Windows 7 or 10, (64-bit) *
	RAM: 					8 GB **
	HDD: 					2 GB free
	Java: 					Java Runtime Environment (JRE) 8, (64-bit) ***
	User Access Rights: 	Administrator privileges ****


*		CTexTools will operate on a 32-bit operating system, but will be constrained to the limits of a 32-bit JRE 
			and will not be able to process files in excess of 20MB.
**		CTexTools will operate on 4GB RAM, but will not be able to process large files. 
			16 GB of system memory is recommended for processing large text files (100 MB or more).
***		CTexTools will operate on a 32-bit JRE, but will not be able to process files in excess of 20MB. 
			To download a 64-bit JRE, go to java.com and download the installer.
****	If one of the included tools does not function as expected, please verify that you are running the 
			software as Administrator with full access rights. Please consult your operating system help file on steps to verify this.
